Animation timing are based on visual weight of the animation element to keep an elegant feel for the webpage

  1. Small sized elements such as buttons, links, icons, droplist,... & simple animations should have a faster animation timing - 0.125s
  2. Regular sized elements such as popups, drawers,... would have a medium animation timing - 0.375s
  3. Large sized elements such as image carousel, drawers,... & complex animations should have a bit longer animation timing - 0.6s
